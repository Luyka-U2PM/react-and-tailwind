
# React + Vite + Tailwind CSS

A modern starter setup for building React applications using Vite and Tailwind CSS.  
The project is lightweight, fast, and ready for further development.

## Technologies

- React 19
- Vite 8.0.10
- Tailwind CSS 4.1.18
- ESLint

## Installation

git clone https://github.com/Luyka-U2PM/react-and-tailwind.git
cd react-and-tailwind
npm install

## Running the project

npm run dev

The application will be available at:
http://localhost:5173

## Production build

npm run build
npm run preview

## Tailwind CSS example

export default function App() {
  return (
     <div className="min-h-screen flex items-center justify-center bg-linear-to-br from-blue-50 to-purple-100">

    </div>
  )
}
